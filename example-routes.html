Reporting API - Example Routes

This document details the RESTful HTTP endpoints implemented on the reporting subdomain to query the Web Analytics database.

Base URL: https://reporting.kazuyamiyata.site/api

Method Endpoint Path Description
GET /sessions Retrieves an array of all recorded user sessions, including static data points (User Agent, Language, Screen Dimensions, etc.).
GET /sessions/{session_id} Retrieves the specific session corresponding to the provided Session ID.
GET /pageviews Retrieves all pageview and pageexit performance payloads, containing load times, TTFB, and navigational enter/leave timestamps.
GET /pageviews/{session_id} Retrieves all pageviews associated with a specific Session ID.
GET /activities Retrieves all raw granular interactions collected by the tracking script, such as mouse movements, clicks, scrolls, keystrokes, and idle periods.
GET /activities/{session_id} Retrieves all granular activities performed by a specific Session ID.
GET /errors Retrieves all JavaScript syntax, resource-loading, and Promise rejection errors dynamically caught on the client side.
